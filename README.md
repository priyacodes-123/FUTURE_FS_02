# 📊 Mini CRM – Client Lead Management System

Successfully completed **Task 2** of the **Future Interns Full Stack Web Development Internship**.

This project is a professional full-stack Mini CRM application developed to manage client leads efficiently with secure admin access, lead tracking, status updates, and follow-up management.

## ✨ Features

• 🔐 Secure Admin Login
• ➕ Add New Leads
• 📋 View Lead Details
• ✏️ Update Lead Status
• ❌ Delete Leads
• 📌 Lead Status Management (New / Contacted / Converted)
• 📝 Follow-up Notes System
• 💾 MongoDB Database Integration
• 📱 Responsive & Professional UI Design

## ▶️ How To Run The Project

1️⃣ Open terminal

2️⃣ Navigate to backend folder

cd backend

3️⃣ Install dependencies

npm install

4️⃣ Start server

npm start

5️⃣ Open frontend/index.html in browser
